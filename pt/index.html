Aplicações de Negócio
com um Núcleo Geoespacial

Quatro décadas de engenharia de software — desde sistemas de negócio em mainframe até plataformas de dados espaciais. Construo aplicações open-source orientadas a dados, onde a inteligência territorial é o diferenciador: back-ends Django + PostGIS, pipelines GIS automatizados e workflows com mapas desenhados para quem realmente os utiliza.

Áreas de Especialização

SIG & Análise Espacial

Análise geoespacial avançada, cartografia e detecção remota — exclusivamente open-source: QGIS, PostGIS, GDAL e Google Earth Engine.

Engenharia de Dados Geoespaciais

Desenho e implementação de pipelines de dados espaciais, processos ETL e bases de dados PostgreSQL/PostGIS.

Programação Espacial

Desenvolvimento de ferramentas geoespaciais, scripts de automação e aplicações de mapeamento web em Python, JavaScript e HTMX — exclusivamente open-source.

Desenvolvimento de Aplicações de Negócio

Desenho e desenvolvimento de aplicações de negócio orientadas a dados com componentes territoriais, construídas com Django, GeoDjango e PostGIS.
